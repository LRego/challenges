# Asking questions
---

Write a program that asks questions to the user and prints them out at the end.

Questions:
- What's your name?
- Where do you live?
- How old are you?

Example usage:
```
$ ruby asking-questions.rb
What's your name?
John
Where do you live?
New York
How old are you?
41
Hello John! You're 41 years old and you live in New York! Awesome!"

```
